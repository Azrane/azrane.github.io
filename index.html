About Me

2023-05-30

I go by Azrane online. I have some socials, my music is on SoundCloud and all your usual streaming places. And I'm a washed-up, previously-aspiring web developer that got left in the dust.

While I don't know the difference between SASS and SCSS or React and Angular, I get by making little things that help me out using my prefered medium: the web browser. Which, btw, holy shit! These days, developer tools remove all the hard work. I would have saved so many hours when I was a teenager.

I'm still a fan of properly-formatted, hand-coded HTML, though. If you're like me, go ahead and view the source code of this page to feel the fuzzies. Don't bother coming at me with the spaces vs. tabs nonsense, though. I'm too old for that these days.

Until I get my projects in order, there's a link up to my blog that I don't post to. And the Mozilla Developer Network has been crucial in learning that everything I learned previously is now a lie and doesn't apply to the semantic web. So there's a link to that too, for me.
